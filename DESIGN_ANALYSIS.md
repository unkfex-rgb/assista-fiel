# Análise de Design - SalveDrew

## Paleta de Cores
- **Verde Vibrante**: #00FF00 (neon green) - Destaque principal, elementos interativos
- **Preto**: #000000 - Fundo principal, contraste
- **Branco**: #FFFFFF - Texto principal, contraste
- **Amarelo**: #FFFF00 - Destaque secundário, badges, números
- **Cinza Escuro**: #1a1a1a - Backgrounds secundários

## Estrutura Visual
1. **Header/Navbar**: Preto com logo, menu horizontal (Escalação, Calendário, Tabela, Posts, Fórum), botões de login/registro
2. **Hero Section**: Imagem de fundo (estádio), overlay escuro, texto grande em branco, CTA button
3. **Seções de Conteúdo**: 
   - Cards com bordas verdes
   - Números/badges em amarelo
   - Fundo preto/cinza escuro
4. **Carousel**: Múltiplos cards deslizáveis
5. **Rodapé**: Links, redes sociais

## Estilo
- Moderno, energético, focado em comunidade
- Tipografia em negrito para destaques
- Muitos elementos interativos
- Design responsivo

## Seções Principais
1. Menu de navegação
2. Hero com CTA
3. Seções de conteúdo (Escalação, Calendário, Patrocinadores)
4. Últimas lives/vídeos
5. Notícias e fórum
6. Rodapé com links e redes sociais
